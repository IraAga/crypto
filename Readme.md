## Description
    This mini project contains the implementation of four cryptographic algorithms. Namely,
        - One-Time Pad
        - Caesar Cipher
        - Scytale Cipher
        - Vigenere's Cipher

## Usage
    > ./demo_otp <message>
    > ./demo_cc <message> <shift_value>
    > ./demo_sc <message> <circumference>
    > ./demo_vc <mesage> <key>
